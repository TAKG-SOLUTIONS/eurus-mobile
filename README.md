# Eurus Mobile Application

Project Eurus aims to create a localized cryptocurrency exchange that would operate under government protection in a regulated and safe manner. The system will be accessible to all types of traders with support for Web and Mobile applications, and a public API. All these systems will facilitate high-security standards to safeguard the users.
